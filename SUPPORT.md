# Suporte

## Antes de abrir uma issue

- consulte o guia principal em [README.md](README.md)
- use o indice tecnico em [docs/README.md](docs/README.md)
- para a aplicação pública, veja [apps/cidadao_de_olho/README.md](apps/cidadao_de_olho/README.md)

## Quando usar cada canal

- bug ou regressao: abra uma issue com o template de bug
- ideia, melhoria ou nova fonte: abra uma issue com o template de feature request
- duvida técnica com contexto reproduzivel: abra uma issue descrevendo objetivo, comando e erro
- vulnerabilidade ou exposicao de credenciais: siga [SECURITY.md](SECURITY.md) e não publique em issue pública

## O que ajuda na triagem

- comando executado
- trecho relevante de log ou stack trace
- configuração sanitizada
- passos mínimos para reproduzir
- impacto esperado e impacto observado
